    Technologie-Stack Übersicht

    Zusammenfassung

    Die Systemarchitektur verwendet einen modernen, Cloud-nativen Technologie-Stack, der für Skalierbarkeit, Zuverlässigkeit und Wartbarkeit ausgelegt ist. Die gewählten Technologien repräsentieren Industriestandards und Best Practices, die eine robuste Enterprise-Performance gewährleisten und gleichzeitig Flexibilität für zukünftiges Wachstum bieten.

    Architekturauswahl

    Komponente Technologie/Produkt Begründung
    Print2CAD Print2CAD - kommerzieller Dienst zur Vektorisierung
    - angeblich viel Erfahrung in der KI Anwendung
    - Zeitersparnis und Risikominimierung Planqualität
    3D Viewer xeokit - Breite Browser-Unterstützung
    - Optimierte 3D-Rendering-Leistung
    - Umfassende API für Integration
    - Unterstützung verschiedener 3D-Formate
    - Integrierte Annotations- und Sharing-Funktionen
    xeokit Browser Demo
    BIM Server BIMserver.org - Industrietaugliche implementierung
    - Bauteile und Gebäude
    -Merge and Match plugins
    BIM Server implementierung open source
    Frontend Framework Angular - Robustes TypeScript-basiertes Framework
    - Umfassende Entwicklungswerkzeuge (Angular CLI)
    - Modulare Struktur für große Anwendungen
    - Integrierte State-Management-Lösungen
    - Starker Enterprise-Fokus
    API Gateway & Auth OAuth2-Proxy mit Nginx - Zentralisierte Authentifizierung
    - Effizientes Load Balancing
    - Hohe Performance und Stabilität
    - Flexible Konfigurationsoptionen
    - Bewährte Sicherheitsstandards
    Datenbank PostgreSQL - Robuste relationale Datenbank
    - Exzellente JSON/JSONB-Unterstützung
    - Hohe Datenkonsistenz
    - Gute Skalierbarkeit
    - Umfassende SQL-Funktionalität
    Workflow Engine Apache Airflow - Asynchronous pipelines
    - Dynamisch skalierbare worker pods
    - Graphisches web UI um alle parallel workflows zu sehen, states zu verfolgen, Probleme diagnostizieren
    Message Broker Apache Kafka - Hohe Skalierbarkeit und Durchsatz
    - Zuverlässige Nachrichtenpersistenz
    - Bewährte Event-Streaming-Plattform
    - Event-Sourcing-Unterstützung
    - Große Community und Enterprise-Support
    Collaboration Hub LobeHub - Lokale db, Interierbar+Plugins
    - Progressive web app
    - Lokale LLMs
    https://lobehub.com/docs/usage/start
    Collaboration Hub Istio - Fortgeschrittene Service-Discovery
    - Verkehrsmanagement
    - Integrierte Sicherheitsfunktionen
    - Beobachtbarkeit
    - Microservices-optimiert
    Container-Orchestrierung Kubernetes - Industriestandard für Container-Orchestrierung
    - Automatische Skalierung
    - Selbstheilende Fähigkeiten
    - Deklarative Konfiguration
    - Große Community-Unterstützung
    CI/CD GitLab CI/CD - Integrierte Pipeline-Funktionalität
    - Container-Registry-Integration
    - Automatisierte Testmöglichkeiten
    - Infrastructure-as-Code-Unterstützung
    - Umfassende DevOps-Funktionen
    Monitoring Prometheus & Grafana - De-facto Standard für Kubernetes-Monitoring
    - Flexible Metrikerfassung
    - Leistungsstarke Visualisierung
    - Umfassende Alarmierungsfunktionen
    - Gute Integration mit anderen Tools
    API-Dokumentation Swagger/OpenAPI - Industriestandard für API-Dokumentation
    - Automatische Dokumentationsgenerierung
    - Interaktives API-Testing
    - Code-Generator-Unterstützung
    - Einfache Integration
